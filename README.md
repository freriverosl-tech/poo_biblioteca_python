# Proyecto POO en Python

Este proyecto implementa:

- Clase Libro (con encapsulamiento)
- Clase Biblioteca
- Clase Estante
- Métodos accesadores y mutadores
- Ejemplo de uso práctico

## Conceptos aplicados

- Programación Orientada a Objetos
- Encapsulamiento
- Composición
- Uso de getters y setters
